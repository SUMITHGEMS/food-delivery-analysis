# 📊 Food Delivery Data Analysis Project

## 🔹 Objective
Analyze food delivery data to understand factors affecting delivery time.

## 🔹 Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- MySQL
- Power BI

## 🔹 Project Steps
1. Data Cleaning using Python
2. Exploratory Data Analysis (EDA)
3. SQL Queries for insights
4. Dashboard creation in Power BI

## 🔹 Key Insights
- Semi-Urban areas have highest delivery time  
- Traffic jams increase delivery delays  
- Electric scooters are faster than motorcycles  

## 🔹 Files in Project
- analysis.py → Python code  
- cleaned_food_delivery.csv → Cleaned dataset  
- sql_queries.sql → SQL queries  
- dashboard.png → Power BI dashboard
